Getting started
===============

Shenfun consists of classes and functions whoose purpose are to make it easy to implement
PDE's in simple tensor product domains. The most important everyday tools are

	* :class:`.TensorProductSpace`
	* :class:`.TrialFunction`
	* :class:`.TestFunction`
	* :class:`.Function`
	* :func:`.div`
	* :func:`.grad`
	* :func:`.project`
